# JWT token 创建终结点

使用Endpoints.UseJwt 可以为应用程序创建验证凭证Token.

# 实例

func main() {
	YoyoGo.CreateDefaultBuilder(func (rb router.IRouterBuilder) {
	         endpoints.UseJwt(rb)
        }.Build().Run()
}

# 配置

创建Token依赖配置节点 application.server.jwt 下: **secret ** , **expires ** 分别用于jwt的密钥和过期时间.

application:
  name: demo
  server:
    # ........
    jwt:
      header: "Authorization"
      secret: "12391JdeOW^%$#@"
      prefix: "Bearer"
      expires: 3
      enable: true

# 访问地址

POST: /auth/token

请求参数：

{ 
   "id":"22" , 
   "name":"yoyo"  
}
