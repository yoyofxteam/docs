# 对象依赖检查终结点

使用Endpoints.UseHealth 可以为应用程序及路由添加健康检查的Endpoint用于Http输出应用状态

# 实例

func main() {
    YoyoGo.CreateDefaultBuilder(func(rb router.IRouterBuilder) {
        endpoints.UseHealth(rb)
    }).Build().Run()       //默认端口号 :8080	
}

# 访问地址

/actuator/health

# 返回值

{
    "status": "UP"
}
